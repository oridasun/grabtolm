Getting a Client ID

  1. Go to console.cloud.google.com and create or select a project.
  2. Enable the Google Drive API in the API Library.
  3. Go to APIs & Services → Credentials.
  4. Click Create Credentials → OAuth 2.0 Client ID (Web application).
  5. Add your site URL to Authorized JavaScript origins.
  6. Copy the Client ID and paste it in the Settings screen.
