Configurar transcripción IA

GrabtoLM usa Groq Whisper para transcribir tus grabaciones en el móvil. Es gratuito — solo necesitas crear una cuenta y obtener tu clave API personal.

Tu clave se guarda solo en este dispositivo y nunca se comparte.

  1. Ve a console.groq.com y crea una cuenta gratuita
  2. En el menú izquierdo, haz clic en API Keys
  3. Haz clic en Create API Key
  4. Copia la clave y pégala aquí:
